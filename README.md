# Real-Time AI-Powered Mock Interview

An AI-powered mock interview platform that evaluates candidates using NLP, speech recognition, and facial emotion detection to provide real-time interview feedback and confidence analysis.

---

## Features

- Dynamic interview question generation
- Speech-to-text transcription using Whisper
- Semantic similarity analysis using SentenceTransformers
- Facial emotion detection using CNN
- Real-time feedback scoring
- Confidence level analysis

---

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- face-api.js

### Backend
- FastAPI
- Python
- PyTorch
- HuggingFace Transformers
- Whisper

---

## Datasets Used

- FER-2013
- CREMA-D
- RAVDESS
- SAVEE

---

## Machine Learning Models Used

- CNN for facial expression recognition
- SentenceTransformer for semantic similarity
- Whisper model for speech-to-text conversion
